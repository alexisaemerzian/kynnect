# App Icons Required

You need to add two PNG icon files to this `/public` folder:

1. **icon-192.png** - 192x192 pixels
2. **icon-512.png** - 512x512 pixels

These icons will be used for:
- PWA installation on mobile devices
- App icon when added to home screen
- Browser tab favicon

## Quick Way to Create Icons:

1. Use your Kynnect logo
2. Go to https://realfavicongenerator.net/
3. Upload your logo
4. Download the generated icons
5. Rename them to `icon-192.png` and `icon-512.png`
6. Place them in this `/public` folder

For now, the app will work without icons, but they're required for a professional PWA experience.
